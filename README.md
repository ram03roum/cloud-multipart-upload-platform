# Cloud Multipart Upload Platform

## Description

A cloud-based multipart upload platform developed and industrialized on Azure, integrating DevOps best practices (CI/CD, Infrastructure as Code, monitoring).

This project enables reliable and performant uploading of large files to Azure Blob Storage using a chunked upload strategy (multipart upload).

## Objectives

- Develop a performant and resilient multipart upload API
- Industrialize deployment through DevOps practices (CI/CD)
- Provision Azure infrastructure using Infrastructure as Code
- Ensure platform reliability and scalability
